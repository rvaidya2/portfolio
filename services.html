High-Impact Projects

Breast Cancer Classification

This project develops a classification model to predict breast cancer diagnosis using the Support Vector Machine (SVM) algorithm, leveraging the Wisconsin Breast Cancer dataset. Achieving a validation accuracy of 96.4% and a test accuracy of 100%, this model demonstrates high efficacy in diagnosis prediction.

  • Scikit-learn
  • Pandas and NumPy
  • Matplotlib

Visualizing Extreme Weather Trends

This project visualizes the highs and lows of temperature data from 2015, highlighting instances where records from the ten-year period (2005-2014) were broken. By assessing these trends, the goal is to determine if extreme weather events are becoming more frequent.

  • Python
  • Data Visualization
  • Jupyter Notebook

Nashville Housing Data Cleaning

This project focuses on cleaning and standardizing the Nashville Housing dataset using SQL queries. Key steps include standardizing date formats, filling in missing addresses, parsing and splitting address details, converting data values, and removing duplicates and unnecessary columns.

  • SQL
  • SQL Server Management Studio
  • Git

Covid Data Analysis

This project conducts in-depth analysis of COVID-19 data, utilizing various SQL techniques and functions. Skills employed include joins, common table expressions (CTEs), temporary tables, window functions, aggregate functions, creating views, and converting data types.

  • Data Exploration
  • Advanced SQL
  • Tableau

Transpostion Cipher

This project implements a transposition cipher for encrypting and decrypting messages. It utilizes key-based encryption to rearrange the characters of a message according to a specific permutation. The project focuses on simplicity, efficiency, and security in cryptography.

  • Python
  • Encryption
  • Cybersecurity

Spreadsheet Clone using React

This project involves creating a web-based spreadsheet application that replicates the Directed Acyclic Graph (DAG) nature found in traditional spreadsheet software. This includes five projects utilizing a range of web development skills.

  • JavaScript/TypeScript
  • MongoDB
  • React
